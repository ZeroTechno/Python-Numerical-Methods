# Numerical Methods in Python

Python implementations of common numerical methods for solving mathematical problems.

## Overview

This repository contains a collection of Python programs for numerical methods topics, including:

- Taylor series approximation
- Interpolation
- Root approximation
- Numerical differentiation
- Numerical integration
- Simultaneous linear systems

Each folder contains a separate program or solution for a specific topic.

## Contents

- `taylor-series/`
- `interpolation/`
- `root-approximation/`
- `numerical-differentiation/`
- `numerical-integration/`
- `simultaneous-linear-systems/`

## Requirements

- Python 3.x

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/ZeroTechno/Python-Numerical-Methods
   ```
2. Go to the project folder:
   ```bash
   cd Python-Numerical-Methods
   ```
3. Run the Python file you want:
   ```bash
   python filename.py
   ```

## Purpose

This repository was created to organize multiple numerical methods projects in one place and keep them easy to browse and maintain.


This project is for academic and personal learning purposes.
